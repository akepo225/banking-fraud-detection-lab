## Decision

**Ready for human v0.6 acceptance review.** All v0.6 child issues (#149–#155) are
merged to main and this review (#156) is the final sign-off artifact.

v0.6 introduces relationship and transaction network analytics as an extension of
the existing tabular data model and case layer. NetworkX is the required graph
layer; Neo4j export stays optional and outside the core learner path and CI;
graph features are explainable and join back to the foundation Progressive data
views; and the private-banking and digital-banking graph notebooks extend — and
do not replace — the v0.3/v0.4 tabular investigation.

The two HITL slices are resolved: #150 (graph-native Detection pattern IDs
`mule_ring` and `circular_funds_movement`) was signed off and merged via PR #166,
and #155 (graph case-library source packs) was signed off and merged via PR #172.
The maintainer's v0.6 acceptance decision on this review (#156) is the final
release gate.

The repository remains private until the separate v0.1 publication gate is
accepted; v0.6 adds no public-release language.
