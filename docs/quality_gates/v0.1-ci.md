# v0.1 CI Quality Gates

This manifest keeps the v0.1 quality gate explicit while the repository is still private. CI uses tiny generated data only and does not create medium or large generated datasets.

## CI Commands

The clean-checkout CI job in `.github/workflows/ci.yml` runs these commands:

- `uv sync --extra dev`
- `uv run ruff check .`
- `uv run pytest`

`uv run pytest` is intentionally unfiltered so every test under `tests/` runs in CI.
