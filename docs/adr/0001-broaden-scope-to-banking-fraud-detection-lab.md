# Broaden Scope To Banking Fraud Detection Lab

Accepted. The curriculum is named **Banking Fraud Detection Lab** and has two first-class tracks: private-banking fraud detection and digital-banking fraud detection. This broadens the original private-banking-only direction because the target learner and interview context include both Swiss private banking and online-bank problems such as scams, mule accounts, e-banking telemetry, digital assets, and brokerage fraud.
