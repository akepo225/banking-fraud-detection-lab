# v0.8 Production-Monitoring Quality Gates

This manifest records where the v0.8 production-monitoring gate evidence lives.
It does not add a separate CI command: the v0.8 module runs under the same
repo-wide quality gates as v0.1 (the unfiltered `uv run pytest` plus
`uv run ruff check .`). `docs/quality_gates/` is v0.1-scoped by convention; this
manifest exists so the v0.8 gate is discoverable rather than implicit.

## CI Commands

The clean-checkout CI job in `.github/workflows/ci.yml` runs these commands
unchanged for v0.8 — there is no v0.8-specific command:

- `uv sync --extra dev`
- `uv run ruff check .`
- `uv run pytest`

`uv run pytest` is intentionally unfiltered, so the v0.8 tests below run in CI.
