# 03_alert_governance

Alert interpretation and governance across both tracks. This module is the
**first governance bridge** in the curriculum: it converts model outputs and
threshold tradeoffs into governance memo drafts with explicit limitations.

It stays the first bridge while deeper layers extend around it — v0.5 case
narratives, v0.6 graph evidence, and the v0.7 model-risk templates. For the full
model-risk treatment (per-alert explanations, model documentation, monitoring
checklist, stakeholder governance memo) continue to
[`07_interpretability_model_risk`](../07_interpretability_model_risk/README.md),
which deepens rather than replaces this module.
